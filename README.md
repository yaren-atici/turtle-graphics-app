# Turtle Graphics App

A Python desktop application built with Tkinter and Turtle graphics that lets you draw geometric and fractal shapes interactively.

## Features

- **Manual control** — Turn left/right and move forward with buttons
- **Square** — Draws a square with small red squares at each corner
- **Star** — Draws a red pentagram
- **Snowflake** — Koch fractal snowflake using recursion
- **Tree 1 & Tree 2** — Two different recursive fractal trees

## Tech Stack

- Python 3
- Tkinter — GUI and button controls
- Turtle — Shape drawing on canvas

## How to run

1. Clone the repository
```bash
   git clone https://github.com/yaren-atici/turtle-graphics-app.git
```
2. Run the app (no extra libraries needed — uses Python standard library)
```bash
   python main.py
```
